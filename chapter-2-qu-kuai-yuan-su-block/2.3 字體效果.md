# 2.3 字體效果

## 斜體字
前後加上一個星號( * )。
```
這是段落中的*斜體字*範例。
```

:arrow\_forward:顯示效果：

這是段落中的*斜體字*範例。

## 粗體字
前後加上兩個星號( * )。
```
這是段落中的**粗體字**範例。
```

:arrow\_forward:顯示效果：

這是段落中的**粗體字**範例。

## 粗斜體
前後加上三個星號( * )。
```
這是段落中的***粗斜體***範例。
```

:arrow\_forward:顯示效果：

這是段落中的***粗斜體***範例。

## 刪除線
前後加上兩個波浪號( ~ )。
```
這是段落中的~~刪除線~~範例。
```

:arrow\_forward:顯示效果：

這是段落中的~~刪除線~~範例。

## 上標字
前後加上一個次方字元( ^ )，或稱為「插入符」，英文為 caret。
```
這是段落中的^上標字^範例。

a^2^ + b^2^ = c^2^
```

:arrow\_forward:顯示效果：

這是段落中的^上標字^範例。

a^2^ + b^2^ = c^2^

正常^上標^
