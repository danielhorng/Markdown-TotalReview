# 2.3 字體效果

## 斜體字
前後加上一個星號( * )。
```
這是段落中的*斜體字*範例。
```

:arrow\_forward:顯示效果：

這是段落中的*斜體字*範例。

## 粗體字
前後加上兩個星號( * )。
```
這是段落中的**粗體字**範例。
```

:arrow\_forward:顯示效果：

這是段落中的**粗體字**範例。

## 粗斜體
前後加上三個星號( * )。
```
這是段落中的***粗斜體***範例。
```

:arrow\_forward:顯示效果：

這是段落中的***粗斜體***範例。

## 刪除線
前後加上兩個波浪號( ~ )。
```
這是段落中的~~刪除線~~範例。
```

:arrow\_forward:顯示效果：

這是段落中的~~刪除線~~範例。

## 第二種斜體字
前後加上一個底線( _ )。
```
這是段落中的_第二種斜體字_範例。
```

:arrow\_forward:顯示效果：

這是段落中的_第二種斜體字_範例。

```
_斜體2_
__斜粗2__
正常^上標^
正常~下標~
++底線++
==螢光標記==
```
