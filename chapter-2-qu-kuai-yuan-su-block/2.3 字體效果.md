# 2.3 字體效果

## 斜體字
前後加上一個星號( * )。
```
這是段落中的*斜體字*範例。
```

:arrow\_forward:顯示效果：

這是段落中的*斜體字*範例。

## 刪除線
前後加上兩個波浪號( ~ )。
```

```

:arrow\_forward:顯示效果：

這是段落中的~~刪除線~~範例。

## 上標字
前後加上一個次方字元( ^ )，或稱為「插入符」，英文為 caret。
```
這是段落中的^上標字^範例。  
a^2^ + b^2^ = c^2^
```

:arrow\_forward:顯示效果：

這是段落中的^上標字^範例。  
a^2^ + b^2^ = c^2^

## 下標字

座標 (x~1~ + y~2~) 及 (X~1~ + Y~2~)

:arrow\_forward:顯示效果：

這是段落中的~下標字~範例。  
座標 (x~1~ + y~2~) 及 (X~1~ + Y~2~)
